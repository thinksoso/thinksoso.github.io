Jupyter notebook极速入门

在学习领域也有一个二八定律 ,即用百分之二十的时间就可以掌握一个领域百分之八十的知识，如果不是专门投身某个领域的话，用百分之二十的时间得到够用的知识显然是很划算的，所以标题是极速入门，重点是快速上手，感受Jupyter notebook的优点。

以前写大作业或者实验报告的时候，都是在ide敲完代码，结果截个图下来粘贴在word里，然后再复制一小块一小块的代码进行说明，工作方法极其原始，而且我因为觉得word设计的不够简洁，操作不够优雅，一般不开word，宁愿开latex写文档。（当然现在用markdown，比如这篇博客）

Jupyter notebook的好处就是可以把代码，运行结果，说明结合在一起

安装（确保你已经安装了anaconda）

conda install jupyter notebook

启动（Jupyter notebook 是以web应用形式运行的，所以实际启动了一个web服务器）

在shell中输入

jupyter notebook

启动所在的目录默认为工作目录

然后就可以用markdown写东西了

并且可以在代码单元格里运行代码
